프로젝트명:
Supabase 기반 바둑 학습 + 학원 운영 SaaS 플랫폼 개발 중.

==================================================
[프로젝트 방향]
==================================================

단순 바둑 문제풀이 앱이 아니라,
최종적으로는 아래를 포함한 통합 플랫폼 목표.

- 바둑 학습
- 승급/승단 심사
- 학원 운영
- 진도 관리
- 오답/복습 시스템
- 시험 생성
- 출석부
- 결제/구독
- 운영 통계

즉:
“바둑 학원 SaaS 플랫폼” 방향.

==================================================
[역할(Role)]
==================================================

1. admin
- 플랫폼 운영자
- 문제/커리큘럼 관리자
- 전체 플랫폼 운영

2. 원장(owner)
- 학원 운영
- 선생/학생 관리
- 학습 현황 관리
- 진도/복습 확인

3. 선생(teacher)
- 담당 학생 관리
- 학습 현황 확인
- 오답/복습 확인

4. 학생(student)
- 문제 풀이
- 오답노트
- 복습
- 학습 진행

==================================================
[현재 주요 메뉴]
==================================================

공통 학습:
- 문제은행
- 학습중

운영:
- 학습관리
- 학원관리

admin 전용:
- 플랫폼 운영

향후 추가:
- 출석부
- 결제 시스템

==================================================
[admin 방향]
==================================================

초기:
- 문제/커리큘럼 관리자

현재:
- 플랫폼 운영 콘솔(admin console) 역할로 확장

현재 플랫폼 운영 구조:
- 학원장(고객) 계정 관리
- 일반 회원 관리
- 구독/결제 관리
- 학원 운영 지표

문제은행은:
모든 역할 공용 학습 시스템.

admin은 관리자 모드에서:
- 문제 생성
- 커리큘럼 관리
- levelGroup 관리
- 문제 순서 편집 (display_order, 카테고리별 공용 순서)
가능.
